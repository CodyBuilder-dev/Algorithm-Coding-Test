# Algorithm-Coding-Test
 입사 코딩테스트 준비를 위한 알고리즘 연습 repo

# 코드 파일명 저장 규칙
    소스-번호-(접미사1-접미사2-...).확장자

명칭 | 예시 | 의미
----|----|----
소스  | `boj`(백준),`pgs`(프로그래머스) 등 | 문제가 수록된 출처 혹은 사이트
번호  | `42627` | 사이트 내에서 문제의 번호
접미사 | `dfs`,`bfs`,`hash` | 어떤 알고리즘/자료구조를 적용해 풀었는지 표시
접미사 | `wrong` | 잘못된 풀이일 경우
접미사 | `note` | 공부해야할 부분이 있는 문제
